# AI Dashcam System

An AI-powered Dashcam system built using Raspberry Pi that provides real-time video recording, intelligent event detection, and vehicle monitoring. The system integrates hardware and software components to enhance driving safety and provide smart surveillance capabilities.

## Features

- Real-time video recording using Raspberry Pi Camera
- AI-based object detection
- Automatic event recording
- GPS tracking support
- GSM/4G connectivity for remote monitoring
- Data storage on SD card
- Easy integration with vehicle systems

## Hardware Components

- Raspberry Pi 4
- Raspberry Pi Camera Module
- GPS Module
- GSM/4G Module
- Micro SD Card
- Power Supply Module
- Vehicle Mount

## Software Requirements

- Python 3
- OpenCV
- TensorFlow / PyTorch
- Raspberry Pi OS
- Git
